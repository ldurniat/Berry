Functions

moveObject (object, x, y)
Moves an object a set distance.

(something that has an X and Y property)

Parameters:

  • object The object to move.
  • x The amount to move the object along the X axis.
  • y The amount to move the object along the Y axis.
round (number, fudge)
Rounds a number.

Parameters:

  • number The number to utils:round.
  • fudge A value to add to the number before rounding. Optional.

Returns:

    The rounded number.
copyPropertiesToObject (objectA, objectB, propertiesToIgnore)
Copies the Properties of one object to another.

For adding to an object that doesn't have 'addProperty' such as a Sprite.

Parameters:

  • objectA The object that has the Properties.
  • objectB The object that will have the Properties coped to it.
  • propertiesToIgnore A list of properties to not add if they exist. Optional.
